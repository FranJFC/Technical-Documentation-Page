Introducción a Python

Python es un lenguaje de programación interpretado, de alto nivel y con una sintaxis clara y sencilla. Es ampliamente utilizado en desarrollo web, análisis de datos, inteligencia artificial, automatización y muchas otras áreas.

Sintaxis Básica

Python utiliza una sintaxis clara y legible. No requiere el uso de llaves para definir bloques de código, sino que usa indentación.

Ejemplo:


    # Esto es un comentario
    print("Hola, mundo!")

Variables y Tipos de Datos

Python es un lenguaje de tipado dinámico, lo que significa que no es necesario declarar el tipo de una variable explícitamente.

Ejemplo:


    x = 5          # Entero
    nombre = "Ana" # Cadena de texto
    pi = 3.14      # Flotante
    es_python = True # Booleano

Estructuras de Control

  • Condicionales

    Python usa if, elif y else para tomar decisiones en el código.

    
    edad = 18
    if edad >= 18:
        print("Eres mayor de edad")
    elif edad == 17:
        print("Casi eres mayor de edad")
    else:
        print("Eres menor de edad")

  • Bucles

    Bucle for

    
    for i in range(5):
    print(i)

    Bucle while

    
    contador = 0
    while contador < 5:
        print(contador)
        contador += 1

Funciones

Las funciones en Python se definen con def. 


    def saludar(nombre):
    return f"Hola, {nombre}!"

    print(saludar("Carlos"))

Listas y Diccionarios

  • Listas

    
    numeros = [1, 2, 3, 4, 5]
    numeros.append(6) # Agregar un elemento
    print(numeros[0]) # Acceder al primer elemento

  • Diccionarios

    
    persona = {"nombre": "Ana", "edad": 25}
    print(persona["nombre"]) # Acceder a un valor

Programación Orientada a Objetos

Python admite POO con clases y objetos.


    class Persona:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad
                            
    def presentarse(self):
        return f"Hola, me llamo {self.nombre} y tengo {self.edad} años."

    p = Persona("Luis", 30)
    print(p.presentarse())

Módulos y Librerías

Python permite reutilizar código mediante módulos y paquetes.


    import math
    print(math.sqrt(16))

Manejo de Archivos

Python facilita la lectura y escritura de archivos.


    with open("archivo.txt", "w") as f:
    f.write("Hola, mundo!")

Excepciones

Python maneja errores con try y except.


    try:
        x = 10 / 0
    except ZeroDivisionError:
        print("No se puede dividir por cero")

Conclusión

Python es un lenguaje versátil y fácil de aprender, ideal tanto para principiantes como para desarrolladores experimentados.
